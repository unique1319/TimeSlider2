# TimeSlider #
## 说明 ##
这是一个非常实用的时间轴，简单大气，支持前后切换和播放功能，支持鼠标点击切换，支持时间轴点击事件回调。

## 示例 ##
### [Demo](https://unique1319.github.io/TimeSlider2) ###

## 使用 ##
```
   var param={};
   param.num=5;
   $(".timeslider").TimeSlider(param,clickback);
   function clickback(time){
    	//TODO your code
   };
```
